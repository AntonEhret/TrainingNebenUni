Javascript Events

What are Events?

Event is something that happens on a web page. It can be done by the browser or the user. Javascript helps us to act whenever such an event takes place.

Example of Events

  • Page was loaded completely by the browser
  • User clicked a button
  • User hovered his mouse over an element

What can you do with these events using Javascript?

We can take an action based on what a user does on a web page. We will use onClick as an example, but the same can be applied for other Javascript events as well. We will apply a couple of more Javascript events in next chapters.
